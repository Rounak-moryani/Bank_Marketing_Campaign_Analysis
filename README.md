# Bank Marketing Campaign Analysis

## Project Overview
This project focuses on analyzing bank marketing campaign data to identify customer behaviors and factors influencing term deposit subscriptions. The analysis helps uncover customer profiles, campaign effectiveness, and seasonal trends to support data-driven marketing strategies.

---

## Objective
The main objectives of this project are:

- Identify customer profiles more likely to subscribe to term deposits.
- Evaluate campaign performance based on contact duration and communication method.
- Analyze the impact of previous campaign outcomes.
- Discover seasonal trends and determine the best time for future campaigns.

---

## Dataset Information
The dataset contains customer and campaign-related information such as:

- Age
- Job Type
- Marital Status
- Education Level
- Housing Loan
- Personal Loan
- Contact Method
- Campaign Duration
- Previous Campaign Outcome
- Subscription Status

Target Variable:
- `y` → Indicates whether the client subscribed to a term deposit (`yes` or `no`).

## Project Workflow

### 1. Data Preparation
- Handled missing and unknown values
- Performed data cleaning and preprocessing
- Converted data types where necessary

### 2. Exploratory Data Analysis (EDA)
- Analyzed distribution of customer demographics
- Explored relationships between features and subscription status
- Visualized patterns using charts and graphs

### 3. Client Profile Analysis
- Studied how job, education, and marital status affect subscriptions
- Identified high-converting customer segments

### 4. Campaign Effectiveness Analysis
- Evaluated the impact of call duration
- Compared communication methods
- Analyzed number of contacts required for successful subscriptions

### 5. Previous Campaign Impact
- Examined how previous campaign outcomes influence current decisions
- Measured the effect of prior customer interactions

### 6. Temporal Analysis
- Identified months with higher subscription rates
- Analyzed weekday-wise campaign performance

---

## Key Insights
- Customers contacted through cellular communication showed better response rates.
- Longer call durations were associated with higher subscription chances.
- Previous successful campaign outcomes increased future conversion probability.
- Certain months showed significantly better campaign performance.
